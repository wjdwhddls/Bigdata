| **특징** | 우편향 분포 (대부분 0~10%, 일부 20%+ 고객 존재) |
| **활용** | 배송 품질 평가, 고객 만족도 지표, 보상 정책 수립 |
| **예시** | delay_rate 26.80% = 이 고객이 받은 상품의 약 1/4이 늦게 도착 |

| 항목 | 내용 |
|---|---|
| **타입** | Float64 |

| **활용** | 주문별 배송 품질 평가, 물류 최적화 |
| **예시** | order_delay_rate 20% = 이 고객의 주문 중 20%가 최소 1개 이상 지연 |

## 💡 실제 활용 사례

### **케이스 1: VIP 고객 식별**
```
조건: total_order_count >= 70회
결과: 상위 10% 고객 약 300명
활용: 우대 배송료 할인, 조기 배송, 전용 고객 센터
```

### **케이스 2: 배송 문제 고객 식별**
```
조건: delay_rate >= 15%
결과: 약 300명 (상위 10% 지연율)
활용: 배송 보상 쿠폰, 배송 문제 모니터링, 배송업체 개선 요청
```

### **케이스 3: 고객 가치도 평가**
```
기준: total_spend (LTV) > 6,000,000원
결과: 약 750명 (상위 25% 고객)
활용: 프리미엄 서비스, 개인화 추천, 이탈 방지 마케팅
```

### **케이스 4: 구매 예측 모델**
```
입력: avg_order_amount, total_order_count, order_amount_std
목표: 다음 3개월 예상 주문 횟수 예측
정확도: std가 낮은 고객일수록 예측 신뢰도 높음
```

## 🎯 결론

**customer_final** 데이터셋은 **고객의 구매 행동과 배송 신뢰도를 통합**하여 볼 수 있는 마스터 고객 프로필입니다.

✅ **주문 빈도** → VIP 고객 식별의 최우선 기준
✅ **누적 구매액** → 고객 생애가치(LTV) 평가
✅ **배송지연율** → 서비스 품질 개선의 근거
✅ **주문금액 편차** → 마케팅 전략의 개인화 정도 판단

이를 통해 **맞춤형 CRM, 정량적 물류 개선, 데이터 기반 마케팅**을 수행할 수 있습니다.
