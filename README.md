# 온라인 새벽배송 고객 분석 및 추천 시스템

온라인 새벽배송 이커머스 플랫폼의 고객 데이터를 분석하여 **고객 세분화**, **개인화 상품 추천**, **매출 예측 모델**을 구축하고, 분석 결과를 **인터랙티브 웹 대시보드**로 시각화한 프로젝트입니다.

---

## 프로젝트 개요

- 3,000명의 고객 구매 데이터를 기반으로 한 종합 분석
- RFM 기반 고객 등급 분류 (VIP / Gold / Silver / Bronze)
- 협업 필터링을 활용한 개인화 상품 추천
- 연관 규칙 분석을 통한 장바구니 상품 연계 추천
- 회귀 모델링을 통한 매출 예측
- React 기반 대시보드로 분석 결과 시각화 및 서비스화

---

## 분석 프로세스

### 1. 데이터 전처리 & 탐색적 데이터 분석 (EDA)

원시 주문 데이터를 정제하고 기초 분석을 수행합니다.

- 결측치 및 중복 데이터 제거
- datetime 파싱 후 **연도, 주차, 주말 여부** 등 시간 피처 추출
- 구매 금액 분포, 카테고리별 매출 추이, 고객별 주문 패턴 등 탐색적 시각화

### 2. 파생변수 생성 (Feature Engineering)

고객의 구매 행동을 다각도로 표현하기 위해 8개의 파생변수를 설계했습니다.

| 파생변수 | 설명 |
|----------|------|
| `weekend_order_ratio` | 전체 주문 중 주말 주문 비율 |
| `category_mix_score` | 구매 카테고리의 다양성 점수 |
| `fresh_food_ratio` | 전체 구매 중 신선식품 비율 |
| `late_delivery_rate` | 배송 지연 발생률 |
| 기타 | 구매 빈도 구간, 평균 주문 금액, Recency 등 |

이 변수들은 이후 RFM 분석 및 예측 모델의 입력 피처로 활용됩니다.

### 3. RFM 고객 세분화 (PCA 가중치 적용)

RFM 분석에 PCA 기반 가중치를 적용하여 고객을 4개 등급으로 분류합니다.

**RFM 지표:**
- **R (Recency)**: 마지막 구매로부터 경과 일수
- **F (Frequency)**: 총 주문 횟수
- **M (Monetary)**: 총 구매 금액

**가중치 산정 (변동계수 기반 PCA):**

| 지표 | 변동계수(CV) | 가중치 |
|------|-------------|--------|
| Recency | 1.1144 | **71.0%** |
| Frequency | 0.2279 | 14.5% |
| Monetary | 0.2266 | 14.4% |

Recency의 변동계수가 가장 높아, 최근 구매 시점이 고객 등급 분류에 가장 큰 영향을 미칩니다.

**등급 분류 기준:**

| 등급 | 종합점수 기준 |
|------|-------------|
| VIP | 80점 이상 |
| Gold | 60 ~ 80점 |
| Silver | 40 ~ 60점 |
| Bronze | 40점 미만 |

### 4. 협업 필터링 추천 (User-User Cosine Similarity)

유사한 구매 패턴을 가진 고객이 구매한 상품을 추천합니다.

**프로세스:**
1. 3,000명 x 58개 상품의 **구매 바이너리 행렬** 생성
2. **연령대별로 분리** (20대, 30대, 40대, 50대, 60대)하여 그룹 내 유사도 계산
3. 각 고객에 대해 **코사인 유사도** 기준 유사 사용자 Top-10 탐색
4. 유사 사용자가 구매했지만 본인이 미구매한 상품 중 **Top-5 추천**

**추천 결과 예시:**

| 순위 | 추천 상품 | 유사도 점수 |
|------|----------|-----------|
| 1 | 국수 | 0.9501 |
| 2 | 땅콩 | 0.6997 |
| 3 | 파프리카 | 0.5542 |
| 4 | 고구마 | 0.4821 |
| 5 | 양배추 | 0.4103 |

### 5. 연관 규칙 분석 (Apriori)

장바구니 분석을 통해 함께 구매되는 상품 조합을 발견합니다.

**방법:**
- mlxtend의 **Apriori 알고리즘**으로 빈발 아이템셋 추출
- 지지도(Support), 신뢰도(Confidence), 향상도(Lift) 기반 연관 규칙 생성
- **연령대별 세분화**하여 총 1,000개 연관 규칙 도출

**연관 규칙 예시:**

| 선행 상품 | 후행 상품 | 지지도 | 신뢰도 | 향상도 |
|----------|----------|--------|--------|--------|
| {콩나물} | {갈치, 파스타} | 13.3% | 52.4% | 1.18 |

> 콩나물을 구매한 고객 중 52.4%가 갈치나 파스타도 함께 구매하며, 이는 무작위 대비 1.18배 높은 확률입니다.

이 규칙은 웹 서비스의 **"함께 구매하면 좋은 상품"** 추천에 활용됩니다.

### 6. 예측 모델링 (Linear Regression & Random Forest)

매출 핵심 지표를 예측하는 모델을 구축합니다.

**종속변수:** 주간 ARPPU (Average Revenue Per Paying User)

**독립변수:** 연령, 가족 수, 회원 여부, 주간 주문 수, 신선식품 비율, 카테고리 다양성, 주말 주문 비율, 배송 지연률 등

**전처리:**
- **VIF(분산팽창계수) 분석**으로 다중공선성이 높은 변수 제거
- **StandardScaler**로 피처 정규화
- 80/20 Train-Test Split

**모델 비교:**

| 모델 | 목적 |
|------|------|
| OLS 회귀 (statsmodels) | 변수별 영향도 해석 (p-value, 회귀계수) |
| Random Forest (scikit-learn) | 예측 성능 극대화 (RandomizedSearchCV 튜닝) |

## 웹 대시보드

분석 결과를 인터랙티브하게 확인할 수 있는 React 기반 웹 애플리케이션입니다.

| 페이지 | 기능 |
|--------|------|
| **홈** | 상품 탐색, 카테고리 필터, 프로모션 배너 |
| **상품 상세** | 상품 정보 + 연관 규칙 기반 "함께 구매하면 좋은 상품" 추천 |
| **마이페이지** | 고객 등급(VIP/Gold/Silver/Bronze), RFM 점수, 협업 필터링 추천 Top-5 |
| **대시보드** | 주요 KPI, 연령/성별 분포, 매출 분포, 배송 지연률 시각화 |
| **세그먼테이션** | 주문 빈도 vs 총 매출 산점도, 연령대별 레이더 차트, 소비 등급 분포 |
| **장바구니** | 연관 규칙 기반 번들 추천, 고객 등급별 할인 적용 |

## 프로젝트 구조

```
Bigdata/
├── analysis/                    # Jupyter 분석 노트북
│   ├── 최종데이터 정제.ipynb     # 데이터 전처리 & EDA
│   ├── 파생변수_분석.ipynb       # 파생변수 생성 및 분석
│   ├── RFM.ipynb                # RFM 고객 세분화
│   ├── 협업필터링.ipynb          # 협업 필터링 추천
│   ├── 연관분석.ipynb            # 연관 규칙 분석
│   ├── 모델개발.ipynb            # 예측 모델링
│   └── 모델개발_월별.ipynb       # 월별 모델 개발
│
├── web/                         # React 웹 애플리케이션
│   ├── src/
│   │   ├── pages/               # 페이지 컴포넌트
│   │   ├── components/          # 공통 컴포넌트
│   │   ├── stores/              # 전역 상태 관리 (Context API)
│   │   └── utils/               # 유틸리티 함수
│   ├── public/data/             # 분석 결과 JSON 파일
│   └── scripts/
│       └── prepare_data.py      # CSV → JSON 변환 스크립트
│
└── data/                        # 원본 및 가공 데이터
```
